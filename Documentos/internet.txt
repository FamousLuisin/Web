O QUE É INTERNET?
 - Internet são redes interligadas por todo o mundo;
 - Uma rede tem como objetivos interligar computadores para fornecer aos usuarios acesso a diversas informações;
 - A internet é feita de cabos que conectam o mundo todo;

REDE DE COMPUTADORES E COMUNICAÇÃO:
  - A internet pode "cair"?
  - Rede de computadores é a conexão de dispositivos para permitir a transmissão de dados;
  - A comunicação tda internet é feita por meio de protocolos (conjuntos de regras);
  - Switch, aparelho que conecta diversos computadores a um unico roteador;

IP E MAC ANDRESS:
  - Internet protocol, é um numero identificador, dado ao seu computador, ou roteador, pelo provedor de internet, 
  no momento em que se conecta a rede;
  - MAC Andress é um numero de série identificador gravado no dispositivo de rede;
  - Atráves do IPque se computador pode enviar e receber dados na internet;

SERVIDORES:
  - Um servidro é um computador equipado com um ou mais processadores, portas de comunicação, softwares, algum sistema de armazenamento de dados como HDs e/ou SSDs;
  - Fornece serviços a uma rede de computadores chamad de cliente. Esses serviços podem ser, por exemplo, hoispedagem de site, provedor de emails, entre outros;

DNS:
  - Os servidores DNS(Domain Name System, ou sitema de nomes de dominio) são os responsaveis por localizar e traduzir para numero de IP os endereços dos sites que digitamos nos navegadores;
  - Os DNS permite que você acesse um site digitando um endereço nominal em lugar de numeros e pontos; 

HTTP:
  - Hypertext Transfer Protocol, sigla HTTP;
  - HTTP é um protocolo de tranferencia de dados;
  - No momento que o usuario digita uma URL em algum navegador, o navegador cria uma requisição HTTP e manda para o servidor correspondente ao IP da URL e então o servidor responde através de HTTP enviando todos os dados necessarios para exibir o site ao usuario no navegador;
  - HTTPS > Hypertext Transfer Protocol Secure, adiciona uma criptografia as informações enviadas ou recebidas de um site;
